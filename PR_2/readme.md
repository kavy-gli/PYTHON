# Pattern Generator and Number Analyzer

## Overview
This is a simple Python program that performs two tasks:

- Generate a star pattern
- Analyze even numbers in a given range

It is made using Python and helps practice:
- Match-case
- Loops
- Conditions
- User input
- Sum calculation

---

## Features

### 1. Pattern Generator
Generates a star pattern like this:

```python
*
**
***
****
*****
```

### 2. Number Analyzer
Takes a starting and ending number and:

- Finds even numbers
- Prints them
- Calculates their sum

Example:

Input:
```python
Start: 1
End: 10
```

Output:
```python
2
4
6
8
10
```

Sum:
```python
30
```
